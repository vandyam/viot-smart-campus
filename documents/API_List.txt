gpio_init()
Prototype
void gpio_init(GPIO_TypeDef *GPIOx, uint32_t pin, gpio_mode_t mode);
Parameters
* GPIOx → GPIO port (e.g., GPIOA, GPIOB)
* pin → Pin number (e.g., GPIO_PIN_5)
* mode → Pin mode
   * GPIO_MODE_OUTPUT_PP_LOW
   * GPIO_MODE_INPUT_PULL_DOWN
   * GPIO_MODE_ANALOG
What it does
Initializes a GPIO pin with the specified mode.
Example
gpio_init(GPIOA, GPIO_PIN_15, GPIO_MODE_OUTPUT_PP_LOW);
________________


gpio_set_iomux()
Prototype
void gpio_set_iomux(GPIO_TypeDef *GPIOx, uint32_t pin, uint8_t function);
Parameters
* GPIOx → GPIO port
* pin → Pin number
* function → Alternate function number
What it does
Configures pin alternate function (used for UART, SPI, etc.).
Example
gpio_set_iomux(GPIOA, GPIO_PIN_8, 1);
________________
